🌸 蜜友 App 圖示產生器

點擊「產生並下載全部圖示」，會自動下載所有尺寸的 PNG 圖示。下載後請放入 icons/ 資料夾。

📋 接下來的步驟

1點擊上方按鈕下載所有圖示（共 5 個 PNG 檔案）
2將下載的 PNG 檔案全部移動到 miyu-pwa/icons/ 資料夾內
3將整個 miyu-pwa/ 資料夾上傳到任一靜態主機：
  • Netlify（推薦）：拖曳資料夾到 app.netlify.com/drop
  • GitHub Pages：推送到 GitHub repo，開啟 Pages 功能
  • 本機測試：執行 python3 -m http.server 8080 後開啟 localhost:8080
4在手機瀏覽器開啟網址，點「加到主畫面」即完成安裝！
